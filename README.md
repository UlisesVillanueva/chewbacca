# WikiStarWars

WikiStarWars is a Python backend and React frontend to show and search StarWars characters.


## Frontend
### Installation

Use npm or yarn
```bash
cd frontend/starwars
```
```bash
yarn install
```
### Run project

```bash
yarn run dev
```




## Backend
### Installation

Use pip3
```bash
cd backend/api
```
```bash
pip3 install -r requirements.txt
```


## Docker
### Run container
```bash
docker-compose up --build
`
